Get Started

Install in four steps

No code, no configuration. Download a file, upload it to Claude, and you are ready.

01
Download the skill

Click the download button on any live skill above to get the .skill file.

02
Open Claude

Go to Customize then Skills in your Claude settings.

03
Upload the file

Click Upload Skill and select the .skill file you downloaded.

04
Start a conversation

Ask your compliance question. Claude will activate the skill automatically.
